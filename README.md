# तुषार बोबडे ट्रान्सपोर्ट अँड अर्थवर्क वेबसाइट

ही एक पूर्णपणे static, GitHub Pages-ready, मराठी व्यावसायिक वेबसाइट आहे. वेबसाइट HTML, CSS आणि JavaScript वापरून तयार केलेली आहे.

## प्रोजेक्ट स्ट्रक्चर

```text
/
  index.html
  about.html
  services.html
  gallery.html
  contact.html
  inquiry.html
  support.html
  /css
    style.css
    responsive.css
  /js
    main.js
  /images
    *.svg
  README.md
```

## वैशिष्ट्ये

- पूर्णपणे मराठीत व्यावसायिक कंटेंट
- responsive mobile-first layout
- sticky navigation bar
- smooth scrolling आणि scroll reveal animations
- animated counters
- gallery filter आणि lightbox preview
- click-to-call आणि WhatsApp integration
- inquiry/contact form validation
- GitHub Pages साठी relative paths

## GitHub Pages वर deploy कसे करावे

1. हा संपूर्ण folder GitHub repository मध्ये upload करा.
2. GitHub repository मध्ये `Settings` > `Pages` मध्ये जा.
3. `Deploy from a branch` निवडा.
4. `main` branch आणि root folder (`/`) निवडा.
5. save केल्यानंतर काही मिनिटांत वेबसाइट live होईल.

## फॉर्मबाबत नोंद

या वेबसाइटमध्ये backend नाही. त्यामुळे संपर्क आणि चौकशी फॉर्म submit केल्यावर थेट WhatsApp message window उघडते.
